Welcome to CommonAPI C++ for SOME/IP!

CommonAPI C++ is a C++ framework for interprocess and network communication which defines a high-level C++ API that can be used for different IPC mechanisms such as SOME/IP. Please visit http://genivi.github.io/capicxx-core-tools/ if you are new to CommonAPI C++.

The CommonAPI C++ SOME/IP binding is based on the standard vsomeip implementation (https://github.com/GENIVI/vsomeip/).

Basic Features

  • High-performance serialization of SOME/IP messages.
